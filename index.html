Data Management

All data is stored in your secure Syrma SGS cloud database (Supabase) and synced in real time across every signed-in user and device. A local browser cache is also kept automatically so the app keeps working briefly if your connection drops.

ID / Password Change Requests

Requests submitted by users from the login screen. Review and resolve them here — no email server is configured, so this in-app list is the system of record. Resolving a request does not change the user's credentials automatically; use Supabase Auth to perform the actual change, then mark it resolved here.
